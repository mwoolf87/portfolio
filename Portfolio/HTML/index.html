Informed-Gamer

Informed-Gamer allows users to search for, and games to favorites list. Users can view release dates, and Metacritic scores of specific games. Soon users will be able to write reviews on games to create a more social environment.

Media-Watchlist

Media-Watchlist is a search engine parents can use when searching for movies for kids! Users create an account either by logging in, or github authentication, from there users can search for movies, view more information such as, parent guide, and where to stream the desired movie and more! They can also add items to favorites which are only viewed by the specific user.
